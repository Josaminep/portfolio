# josamine-portfolio

Quick start:

1. Install dependencies
   npm install

2. Start dev server
   npm run dev

3. EmailJS
   - Create account at https://emailjs.com
   - Create a Service, Template, and get your Public Key
   - Copy them into a .env file (use .env.example)

4. Build
   npm run build

5. Deploy
   - Push to GitHub and deploy to Vercel or Netlify (select build command `npm run build`, publish `dist`)
